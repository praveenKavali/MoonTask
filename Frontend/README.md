# Moon Task

**Task Service Project:** A full stack application for task management. Where you can
store task and do some operations for those tasks.

## ✨ Features 

**Home page:** Contains buttons for registration and user login for creating and login to their user accounts.
**Task page:** Contains all task that you stored.
**Search bar:** For searching through tasks to find for task from list of tasks.
**Filter Bar:** You can filter tasks based on their status and priority.
**User Operation:** Contains user related operation features(eg: update, delete) account.

## 🚀 Tech Stack
**-Framework:** React
**-Libraries:** Axios, SWR
**-Styling:** Css module
